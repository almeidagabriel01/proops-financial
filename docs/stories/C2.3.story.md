## Context

O dashboard do Finansim já mostra receitas, despesas, saldo, projeções e gráficos por categoria. Mas falta um número único que responda "estou indo bem financeiramente?". O score de saúde financeira é esse número — de 0 a 100, com um badge visual e um mini gráfico de evolução.

**Por que isso importa:**
- GuiaBolso (fintech BR, falida em 2022) tinha esse score como feature principal — era citado em reviews como "o motivo de voltar ao app todo mês"
- Copilot Money tem "Financial Health Score" e é uma das features mais elogiadas internacionalmente
- Nenhum concorrente brasileiro ativo oferece hoje

O score é calculado **server-side** a partir de dados que já existem no banco. Não requer nenhuma entrada extra do usuário — apenas dados de transações, orçamentos e metas já registrados.

**Fórmula (0-100):**

| Componente | Peso | Como calcular |
|-----------|------|---------------|
| Taxa de poupança | 40% | `(receita - despesa) / receita × 100`, mapeado em 0-100 (0% ou negativo → 0; ≥30% → 100) |
| Cumprimento de orçamentos | 30% | `% de orçamentos ativos não estourados no mês` (sem orçamentos ativos → 50 pontos neutros) |
| Progresso de metas | 20% | `média do progresso (%) das metas ativas`, ponderado por proximidade do prazo |
| Diversificação de gastos | 10% | Penaliza se uma única categoria > 60% das despesas (`1 - max(category_share - 0.6, 0) / 0.4 × 100`) |

**Badges visuais:**
- 0-25: Crítico (vermelho)
- 26-50: Regular (laranja)
- 51-75: Bom (azul)
- 76-100: Excelente (verde)

O score é calculado para o **mês selecionado** no dashboard. O mini gráfico mostra os últimos 3-6 meses armazenados na tabela `health_score_history`.

### Cálculo do score (TypeScript)

```typescript
// src/lib/health-score/calculate.ts

interface ScoreComponents {
  savingsRateScore: number;    // 0-100, peso 40%
  budgetComplianceScore: number; // 0-100, peso 30%
  goalsProgressScore: number;  // 0-100, peso 20%
  diversificationScore: number; // 0-100, peso 10%
}

function calculateFinalScore(components: ScoreComponents): number {
  return Math.round(
    components.savingsRateScore * 0.4 +
    components.budgetComplianceScore * 0.3 +
    components.goalsProgressScore * 0.2 +
    components.diversificationScore * 0.1
  );
}

function getSavingsRateScore(income: number, expenses: number): number {
  if (income <= 0) return 0;
  const rate = (income - expenses) / income;
  if (rate <= 0) return 0;
  if (rate >= 0.3) return 100;
  return Math.round((rate / 0.3) * 100);
}

function getBudgetComplianceScore(budgets: BudgetWithSpent[]): number {
  if (!budgets.length) return 50; // neutro quando não há orçamentos
  const compliant = budgets.filter(b => b.spent <= b.monthly_limit).length;
  return Math.round((compliant / budgets.length) * 100);
}

function getGoalsProgressScore(goals: Goal[]): number {
  const active = goals.filter(g => g.status === 'active');
  if (!active.length) return 50; // neutro quando não há metas
  const avg = active.reduce((sum, g) => {
    return sum + Math.min(g.current_amount / g.target_amount, 1);
  }, 0) / active.length;
  return Math.round(avg * 100);
}

function getDiversificationScore(categoryTotals: Record<string, number>, totalExpenses: number): number {
  if (totalExpenses === 0) return 100;
  const maxShare = Math.max(...Object.values(categoryTotals)) / totalExpenses;
  if (maxShare <= 0.6) return 100;
  return Math.max(0, Math.round((1 - (maxShare - 0.6) / 0.4) * 100));
}

function getBadge(score: number): 'critico' | 'regular' | 'bom' | 'excelente' {
  if (score <= 25) return 'critico';
  if (score <= 50) return 'regular';
  if (score <= 75) return 'bom';
  return 'excelente';
}
```
